Экспорт из Telegram

Telegram Desktop → ваш чат → ⋮ → «Export chat history» → формат HTML. Без ботов, без сторонних сервисов — только вы и ваш файл.

💡Подойдёт любой личный чат: с парнём, женой, мужем, лучшим другом.

Пример

✓ messages.html
